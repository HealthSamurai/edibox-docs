# Getting Started with Edibox

Edibox provides healthcare EDI transaction processing for X12 and HL7 messages, enabling seamless healthcare data exchange.

## What is Edibox?

Edibox provides:

- **X12 Processing** - Parse and generate X12 EDI transactions (837, 835, 270, 271, etc.)
- **HL7 v2 Support** - Handle HL7 v2 messages (ADT, ORM, ORU, etc.)
- **FHIR Mapping** - Convert between EDI formats and FHIR resources
- **Validation** - Validate transactions against implementation guides
